>
> ###### 이 책의 저자는 Ubuntu 22.04, kernel version 5.15를 사용한다.
> 
> 
>   > 다른 배포판이나 커널 버전을 사용하는 경우 여러 패키지와 종속성 간의 호환성 문제를 해결해야 한다.
